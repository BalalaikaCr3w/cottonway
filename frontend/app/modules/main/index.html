Игра в области компьютерной безопасности RUSCRYPTO PKI QUEST представляет собой набор из двух типов заданий: "Викторина" и "Квест". Задания типа "Викторина" требуют нахождения строки — так называемого флага — на основе предложенных данных, за решение заданий данного типа игроку начисляются очки. Задание типа "Квест" требует выполнения игроком предопределенной последовательности действий. Для прохождения некотрых шагов "Квеста" игроку может потребоваться потратить очки, заработанные при решении заданий "Викторины".

Победителем в игре считается игрок, выполнивший задание типа "Квест" или продвинувшийся в решении задания "Квест" дальше остальных. В случае, если несколько игроков решили задание "Квест" или имеют одинаковый прогресс в решении, победителем считается заработавший наибольшее количество баллов. В случае равенства баллов и прогресса решения задания "Квест" у нескольких игроков, победителем считается первый набравший данное количество баллов, если задание типа "Квест" решено полностью, и первый достигший данного прогресса решения задания типа "Квест" в противном случае.

Для победителя игры один из золотых спонсоров конференции Рускрипто приготовил приятный сюрприз. Активные участники игры получают поощрительные призы от организаторов игры.

В ходе выполнения заданий запрещается производить вредоносные действия по отношению к другим участникам игры и по отношению к инфраструктуре игры.
