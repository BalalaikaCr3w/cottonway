Игра в области компьютерной безопасности RUSCRYPTO PKI QUEST представляет собой набор из двух типов заданий: "Викторина" и "Квест". Задания типа "Викторина" требуют нахождения строки — так называемого флага — на основе предложенных данных, за решение заданий данного типа игроку начисляются очки. Задание типа "Квест" требует выполнения игроком предопределенной последовательности действий. Для прохождения некотрых шагов "Квеста" игроку может потребоваться потратить очки, заработанные при решении заданий "Викторины".

Победителем в игре считается игрок, выполнивший задание типа "Квест" и заработавший наибольшее количество баллов. В случае равенства баллов у нескольких игроков, победителем считается первый закончивший решение задания типа "Квест".

В ходе выполнения заданий запрещается производить вредоносные действия по отношению к другим участникам игры и по отношению к инфраструктуре игры.
